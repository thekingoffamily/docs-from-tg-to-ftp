Данный проект создавался для нужд внутри компании.
Задача передавать определённые PDF документы на вход в telegram bot с уникальным номером документа, далее идёт подключение к FTP серверу.
После успешного подключения происходят определённые проверки и если всё удачно, то файл заливается на сервер в определённую папку. И так по кругу.
